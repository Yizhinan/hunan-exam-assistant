# 知识库素材目录

手动放置 RAG 资料到这里，然后运行 `python scripts/import_knowledge.py` 批量导入。

## 目录说明

| 目录 | 对应 doc_type | 放什么 |
|------|-------------|--------|
| `exam/` | exam | 湖南省考历年真题、标准答案、解析 |
| `policy/` | policy | 湖南省政府工作报告、三高四新、乡村振兴等政策文件 |
| `news/` | news | 湖南时政热点文章 |
| `model/` | model | 高分申论范文 |

## 支持的文件格式

- **PDF** (`.pdf`) — 真题试卷、官方文件
- **Markdown** (`.md`) — 整理的笔记、范文
- **纯文本** (`.txt`) — 新闻、文章

## 命名建议

```
exam/2024_湖南_申论_真题.pdf
exam/2024_湖南_申论_解析.pdf
exam/2023_湖南_行测_常识判断_真题.md
policy/2025_湖南省政府工作报告.md
policy/三高四新_政策汇编.pdf
news/2025年6月_湖南时政汇总.md
model/申论高分范文_乡村振兴类.md
```
